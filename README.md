# Snake Classic

אתר סטטי קטן של משחק Snake קלאסי. אין צורך בהתקנות או build.

## הרצה מקומית

פשוט פותחים את הקובץ `index.html` בדפדפן.

## שיתוף עם אחרים

הפרויקט מוכן לעלות כמו שהוא לכל שירות של static hosting:

1. GitHub Pages
2. Netlify
3. Vercel

כל הקבצים נמצאים בשורש הפרויקט:

- `index.html`
- `style.css`
- `script.js`

## העלאה ל-GitHub Pages

1. צור repository ב-GitHub
2. העלה את שלושת הקבצים האלה
3. ב-GitHub היכנס ל-`Settings` > `Pages`
4. בחר deploy מה-branch הראשי
5. GitHub ייצור לך כתובת קבועה שאפשר לשלוח

## מה כולל האתר

- משחק Snake קלאסי
- תמיכה במקלדת ובמובייל
- שמירת שיא מקומי בדפדפן
- עיצוב רספונסיבי
